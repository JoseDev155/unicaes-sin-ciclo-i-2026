## **Tema: 1. Presentación sobre Tipos de Sistemas Informáticos**
### ¿QUÉ ES UN SISTEMA INFORMÁTICO?
#### UN SISTEMA INFORMÁTICO ES:

Una técnica que permite el almacenamiento y el proceso de información, para lo cual se vale de un grupo de elementos que se relacionan entre sí.

Estos elementos son:
* El hardware
* El software
* El usuario

#### COMPONENTES DE UN SISTEMA INFORMÁTICO

**COMPONENTE FÍSICO:**
* El hardware.

**COMPONENTE LÓGICO:**
* El software del sistema informático
* Conformado por:
  * En primera instancia por el firmware
  * El sistema operativo
  * El sistema de gestión de datos

#### LOS SISTEMAS INFORMÁTICOS SE CLASIFICAN EN:

**Sistemas inalámbricos**
* ESS
* GDSS
* CSCWS

**Sistemas empresariales**
* Sistemas expertos
* Sistemas de soporte de decisiones
* Sistemas de información administrativa

* Sistemas de trabajo de conocimiento
* Sistemas de automatización de oficinas

**Sistemas Web y de comercio electrónico**
* Sistemas de procesamiento de transacciones

##### SISTEMAS DE APOYO A LA TOMA DE DECISIONES:
Conocido como **Sistema de soporte a la decisión** o DSS (Decision Support System), se trata de un sistema basado en computadoras diseñado
para ser utilizado por una gerencia o gerencia de área.

* Ayudan a la gerencia en el proceso de tomar una decisión para resolver problemas.
* Permitir diagramar directrices para seleccionar la mejor opción.
* Predecir futuros escenarios para afrontar nuevos desafíos.

##### SISTEMA DE CONTROL DE PROCESOS DE NEGOCIO:
Los **Business Process Management (BPM)** son sistemas encargados de **monitorear, controlar y gestionar procesos industriales o de negocio**.

* Utilizan sensores electrónicos conectados a computadoras.
* Permiten supervisar directamente el funcionamiento de la maquinaria o procesos.
* Su objetivo es garantizar que los procesos se ejecuten **de manera eficiente y controlada**.

##### SISTEMAS DE COLABORACIÓN EMPRESARIAL
Los **Enterprise Resource Planning (ERP)** son sistemas informáticos que permiten **gestionar y centralizar la información de una empresa**.

* Integran diferentes áreas de la organización.
* Facilitan el manejo de grandes volúmenes de información.
* Mejoran la coordinación entre departamentos y procesos empresariales.

##### SISTEMAS DE INFORMACIÓN EJECUTIVA
Los **Executive Information Systems (EIS)** proporcionan **acceso rápido a información crítica para la toma de decisiones**.

* Integran datos de **fuentes internas y externas**.
* Presentan la información en **formatos visuales y fáciles de interpretar**.
* Están orientados a **directivos y altos ejecutivos**.

##### SISTEMAS DE PROCESAMIENTO DE TRANSACCIONES
Los **Transaction Processing Systems (TPS)** gestionan las **operaciones diarias de una organización**.

* Registran y procesan las transacciones realizadas durante la jornada.
* Almacenan información operativa fundamental.
* Son esenciales para el funcionamiento básico de cualquier empresa.

##### SISTEMAS DE INFORMACIÓN DE GESTIÓN
Los **Management Information Systems (MIS)** recopilan y procesan datos para apoyar la **gestión y toma de decisiones administrativas**.

* Utilizan datos generados por otros sistemas, especialmente los **TPS**.
* Transforman la información en **reportes y análisis**.
* Ayudan a supervisar y controlar los procesos de la organización.

#### ¿CÓMO FUNCIONA UN SISTEMA INFORMÁTICO?

Su misión es la de ofrecer a una compañía un sistema de control de fabricación, un sistema de contabilidad, un sistema de distribución y
logística.

Esto por medio de computadoras conectadas entre sí, en los distintos departamentos de la empresa comparten información, datos e incluso
software o equipamiento periférico, un ejemplo sencillo puede ser una impresora en red.

## **Tema 2: Construyendo Propuesta Proyecto de Software**
### Definición del proyecto SI para el Ciclo
#### Una vez definidos los Equipos de trabajo

* Participantes por grupo de 3-5
* Definir 1 propuesta de la viabilidad del proyecto a desarrollar
* Documentar la propuesta en el formato publicado en MOODLE

#### Tener en cuenta los siguientes parámetros en el Sistema Informático

* Debe ser completo y no manejar un solo módulo, puesto que debe implementar las fases del ciclo de un Software.
* Debe tener elementos o funciones innovadoras y en tendencia.
* No se recomienda reciclar software de materias anteriores.
* Lo ideal es que realicen un SI de una empresa real a la que ustedes le proveerán dicho servicios de Desarrollo e implementación del Software.
* Si se determina que es un Software extraído de algún sitio, se anula automáticamente su proyecto y deberá iniciar uno nuevo.
* El SI localmente funcional deberá compilarse y generar los .exe (cuando aplique). Pues se espera la proyección funcional teniendo en cuenta el contexto real.
* Si es en la Web deberá alojarlo en un host para asegurar su funcionamiento real.

#### Es de Suma Importancia:

1. Investigar y elegir el lenguaje de programación apropiado para su SI.
2. Investigar y elegir gestor de Bases de Datos que sea viable a las conexiones requeridas para su SI.
3. Elegir desde el principio la herramienta de testeo para llevar un seguimiento del SI trabajado, que le permita administrar su proyecto.
